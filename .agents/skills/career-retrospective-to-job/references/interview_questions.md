# Interview Questions

Use questions in rounds. Ask only the next useful group instead of sending the whole list at once.

## Round A: Goals And Constraints

- 你最想做哪类工作？
- 你正在考虑哪些岗位？
- 哪些岗位明确不考虑？
- 城市、行业、公司类型有什么偏好或限制？
- 现在最重要的是 offer、方向、学习、薪资、稳定性还是平台背书？
- 实习、秋招、全职、远程、毕业时间等硬约束是什么？

## Round B: Motivation And Values

- 你为什么想走这个方向？
- 哪类工作会让你持续有精力？
- 哪类工作会让你明显消耗？
- 你希望你的工作帮助用户或组织解决什么问题？
- 3-5 年后你希望自己变成什么样的人？

## Round C: Project Depth

For each key project:

- 它解决了什么真实问题？
- 真实用户是谁？
- 你个人具体做了哪些事情？
- 用了什么工具、方法、技术或流程？
- 有什么能验证结果的数字、反馈或交付物？
- 哪些地方失败、暂停或没有完成？
- 如果继续做，你下一步会怎么改？
- 面试时哪些细节你能讲清楚，哪些需要回避或补证据？

## Round D: Skill Boundaries

Ask the user to classify each skill:

- 可独立完成
- 熟悉，可在 AI 或文档辅助下完成
- 了解，做过 demo
- 只接触过，不适合主打

## Round E: Truth And Risk

- 哪些经历是正式实习？
- 哪些是个人项目、课程项目或合作项目？
- 哪些数据有证据，哪些只是估算？
- 哪些公司名、客户名、业务数据不能公开？
- 哪些经历容易被误解为夸大？
- 哪些未来计划、在投论文、未上线产品必须谨慎表达？
